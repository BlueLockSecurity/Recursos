# Recursos y Evidencias: Trabajo Final

Este repositorio almacena todos los recursos técnicos, scripts y artefactos de apoyo generados durante el proyecto final del curso **Anti-Hacking y Nuevas Tendencias de Seguridad (1ASI0665)**.

El propósito de este repositorio es centralizar las evidencias y herramientas utilizadas, sirviendo como un anexo técnico para el informe principal del proyecto.

## 📂 Contenido del Repositorio

Este repositorio incluye:

* **/scripts:** Todos los scripts personalizados (Python, Bash, etc.) creados para automatizar tareas de reconocimiento, escaneo o explotación.
* **/payloads:** Muestras de código de payloads, web shells y reverse shells utilizadas durante las pruebas de acceso.
* **/evidencias:** Capturas de pantalla, volcados de tráfico (`.pcap`), registros (logs) y salidas de herramientas (Nmap, Burp Suite, sqlmap, etc.) que validan los hallazgos.
* **/wordlists:** Diccionarios personalizados usados en ataques de fuerza bruta.
* **/notas:** Archivos de texto con notas de reconocimiento, enumeración y hallazgos preliminares.
